## Challenge 4: Remove Middle Value

### Challenge Description
This challenge aims to remove the middle value from a given dataset, such as an array or a linked list. If the dataset has an even number of elements, the function should remove the lower middle value.

### Approach:
- **Step 1:** Determine the length of the dataset.
- **Step 2:** Identify the middle index.
- **Step 3:** Remove the middle value efficiently.
- **Step 4:** Return the updated dataset.
